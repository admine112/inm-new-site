# Инструкция: Как обновить мета-теги для страницы "Состав Инмунофлама"

## Страница: https://inmunoflam.com.ua/sostav-inmunoflama

---

## Способ 1: Через админку сайта (рекомендуется)

### Шаг 1: Войти в админку
1. Открыть https://inmunoflam.com.ua/admin (или ваш адрес админки)
2. Ввести логин и пароль

### Шаг 2: Найти страницу
1. Перейти в раздел "Контент" или "Страницы"
2. Найти страницу с названием "Состав Инмунофлама" или alias "sostav-inmunoflama"
3. Нажать "Редактировать"

### Шаг 3: Обновить мета-теги

**Title (заголовок для Google):**
```
Инмунофлам: мощная природная поддержка Т-клеточного иммунитета и профилактика заболеваний
```

**Meta Description (описание для Google):**
```
Инмунофлам — растительный иммуномодулятор с более чем 70 активными компонентами. Иммуномодулирующее, антиоксидантное и противовоспалительное действие, подтвержденное клиническими исследованиями.
```

**H1 (главный заголовок страницы):**
```
ЧТО СОБОЙ ПРЕДСТАВЛЯЕТ ИНМУНОФЛАМ
```

### Шаг 4: Сохранить
1. Нажать кнопку "Сохранить" или "Обновить"
2. Проверить изменения на сайте

## Проверка после обновления

### 1. Проверить на сайте
Открыть https://inmunoflam.com.ua/sostav-inmunoflama и проверить:
- В заголовке браузера должен быть новый Title
- На странице должен быть H1: "ЧТО СОБОЙ ПРЕДСТАВЛЯЕТ ИНМУНОФЛАМ"

### 2. Проверить в коде страницы
Нажать Ctrl+U (просмотр кода страницы) и найти:
```html
<title>Инмунофлам: мощная природная поддержка...</title>
<meta name="description" content="Инмунофлам — растительный иммуномодулятор...">
<h1>ЧТО СОБОЙ ПРЕДСТАВЛЯЕТ ИНМУНОФЛАМ</h1>
```

### 3. Проверить в Google
Через 1-2 недели проверить, как страница отображается в поиске Google.
