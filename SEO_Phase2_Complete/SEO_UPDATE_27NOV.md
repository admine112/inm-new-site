## Как это внедрить?

### Вариант 1: Через базу данных (рекомендуется)

Запустить SQL-скрипт `update_sostav_meta.sql`:

```sql
UPDATE content 
SET 
    title = 'Инмунофлам: мощная природная поддержка...',
    description = 'Инмунофлам — растительный иммуномодулятор...',
    h1 = 'ЧТО СОБОЙ ПРЕДСТАВЛЯЕТ ИНМУНОФЛАМ'
WHERE alias = 'sostav-inmunoflama';
```

### Вариант 2: Через админку сайта

1. Зайти в админку
2. Найти страницу "Состав Инмунофлама"
3. Вставить новые Title, Description, H1
4. Сохранить
