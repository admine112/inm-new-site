# Пошаговая инструкция: Как выполнить SQL-запрос

## Вариант 1: Через phpMyAdmin (самый простой)

### Шаг 1: Войти в phpMyAdmin
1. Открыть браузер
2. Перейти на адрес вашего хостинга + `/phpmyadmin`
   - Например: `https://inmunoflam.com.ua/phpmyadmin`
   - Или через панель хостинга (cPanel, ISPmanager и т.д.)
3. Ввести логин и пароль от базы данных

### Шаг 2: Выбрать базу данных
1. В левой колонке найти базу данных `inmunoflam`
2. Кликнуть на нее

### Шаг 3: Открыть SQL-редактор
1. Вверху найти вкладку **"SQL"**
2. Кликнуть на нее

### Шаг 4: Вставить запрос
1. В большое текстовое поле вставить этот код:

```sql
UPDATE content 
SET 
    title = 'Инмунофлам: мощная природная поддержка Т-клеточного иммунитета и профилактика заболеваний',
    description = 'Инмунофлам — растительный иммуномодулятор с более чем 70 активными компонентами. Иммуномодулирующее, антиоксидантное и противовоспалительное действие, подтвержденное клиническими исследованиями.',
    h1 = 'ЧТО СОБОЙ ПРЕДСТАВЛЯЕТ ИНМУНОФЛАМ'
WHERE alias = 'sostav-inmunoflama';
```

### Шаг 5: Выполнить запрос
1. Нажать кнопку **"Вперед"** или **"Go"** (внизу справа)
2. Должно появиться сообщение: "1 row affected" или "Затронута 1 строка"

### Шаг 6: Проверить результат
1. Вставить этот запрос в то же поле:

```sql
SELECT id, name, alias, title, description, h1 
FROM content 
WHERE alias = 'sostav-inmunoflama';
```

2. Нажать **"Вперед"**
3. Должна появиться таблица с обновленными данными

## Вариант 2: Через командную строку (для продвинутых)

### Если у вас есть SSH-доступ к серверу:

```bash
# Подключиться к серверу
ssh ваш_логин@inmunoflam.com.ua

# Войти в MySQL
mysql -u root -p

# Ввести пароль от MySQL

# Выбрать базу данных
USE inmunoflam;

# Выполнить запрос (скопировать весь блок)
UPDATE content 
SET 
    title = 'Инмунофлам: мощная природная поддержка Т-клеточного иммунитета и профилактика заболеваний',
    description = 'Инмунофлам — растительный иммуномодулятор с более чем 70 активными компонентами. Иммуномодулирующее, антиоксидантное и противовоспалительное действие, подтвержденное клиническими исследованиями.',
    h1 = 'ЧТО СОБОЙ ПРЕДСТАВЛЯЕТ ИНМУНОФЛАМ'
WHERE alias = 'sostav-inmunoflama';

# Проверить результат
SELECT id, name, alias, title, description, h1 
FROM content 
WHERE alias = 'sostav-inmunoflama';

# Выйти
exit;
```

## Проверка после выполнения

### 1. Проверить в базе данных
Выполнить запрос:
```sql
SELECT title, description, h1 FROM content WHERE alias = 'sostav-inmunoflama';
```

Должно показать:
- **title:** Инмунофлам: мощная природная поддержка...
- **description:** Инмунофлам — растительный иммуномодулятор...
- **h1:** ЧТО СОБОЙ ПРЕДСТАВЛЯЕТ ИНМУНОФЛАМ

### 2. Проверить на сайте
1. Открыть https://inmunoflam.com.ua/sostav-inmunoflama
2. Посмотреть в заголовок браузера (вкладка)
3. Нажать Ctrl+U (просмотр кода страницы)
4. Найти `<title>` и `<meta name="description">`

## Если что-то пошло не так

### Ошибка: "Table 'content' doesn't exist"
**Решение:** Проверьте название таблицы в вашей базе данных. Возможно, она называется по-другому (например, `pages`, `articles` и т.д.)

### Ошибка: "Unknown column 'alias'"
**Решение:** Проверьте, как называется поле с URL страницы. Возможно, это `url`, `slug` или другое название.

### Ошибка: "0 rows affected"
**Решение:** Страница с таким alias не найдена. Проверьте правильность alias в базе данных.
